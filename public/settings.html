Webhook configuration

Webhook URL

POST

Send POST requests to this endpoint with optional URL parameters

Available URL Parameters:
• customer_name={string} - Customer's full name
• dealership_name={string} - Dealership name (default: America First)

Request Format

Content-Type: application/json
x-api-key: sms-webhook-secure-2024
{
  "transactionId": "TEST-1234567890",
  "customerPhone": "+14155552671",
  "customerName": "John Doe",     // Optional, or use URL param
  "dealershipName": "ABC Motors",  // Optional, or use URL param
  "amount": 299.99                 // Optional
}

URL parameters override body values if both are provided

API Key

sms-webhook-secure-2024

Include this key in the x-api-key header for authentication
